RejsonIPC¶

class dasha.web.extensions.ipc_backends.rejson.RejsonIPC(label)[source]¶

Bases: object

This class manages a JSON object in redis.

The actual object stored in the redis db under redis_key looks like:

{
    '_ipc_obj': {'a': 'b'},
    '_ipc_meta': {
        'key': 'redis_key'
        'rev': 0,
        'created_at': xxxx,
        'updated_at': yyyy,
        }
    }
