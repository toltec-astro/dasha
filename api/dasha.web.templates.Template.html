Template¶

class dasha.web.templates.Template(parent=None)[source]¶

Bases: dasha.web.templates.IdTree

An abstract class that defines an encapsulated entity around a functional group of dash components and the callbacks.

One can do arbitrary nesting of the template through the child factory function, which setup the created instance as the child.

The template instance serve as a lazy evaluation proxy around the wrapped Dash components. Modification to relevant Dash component properties are registered as kwargs that will pass to the actual component constructor when the layout property is queried.
